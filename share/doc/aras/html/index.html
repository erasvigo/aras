ARAS is a free broadcast automation system. Project goals are:

  • To be reliable
  • To be highly configurable
  • To be multi-platform
  • To be a growing project

Source code, binaries and deb packages for x86-64 GNU/Linux are available. Current release is ARAS-4.5. It contains three applications:

ARAS Daemon-4.5 is a scheduler with an integrated media player, without graphical interface and without GTK+ requirements.
ARAS Player-4.5 is a scheduler with an integrated media player and with graphical interface.
ARAS Recorder-4.5 is an audio recorder that can be useful to record the defined blocks.

You can learn more about ARAS by reading its documentation.
